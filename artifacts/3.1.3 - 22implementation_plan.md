# Phase 4: The Functional Pivot (Generic Engine)

## Goal Description
Refactor the entire Brinkmanship directory from a hard-coded modern geopolitical simulator into a Theme-Agnostic, Functional State Engine. This involves implementing a Redux-style immutable state machine, decoupling scenario strings into JSON configs, upgrading the Chronos engine to fractional Gaussian math, and supporting multiple scenario layers (Modern, Cyberpunk, Fantasy).
